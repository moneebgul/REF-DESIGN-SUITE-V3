Method & references
UA = 1 / ( 1/(η₀·h_air·A_o) + 1/(h_ref·A_i) ) ; NTU = UA/(ṁ_air·c_p) ; ε = 1 − e^(−NTU) [phase-change, C_r≈0]
approach = Q / (ε·ṁ_air·c_p) → T_e = T_air − approach (evap) ; T_c = T_air + approach (cond)
static coil: approach = Q / (η₀·h_air·A_o) [air = constant-temperature reservoir]

Air-side h from a Colburn j-factor for plain fin-and-tube (j ≈ 0.29·Re_Dc^−0.36, representative); fin efficiency η_f = tanh(mL)/mL, m = √(2h/k_fin·t_fin); surface efficiency η₀ = 1 − (A_fin/A_o)(1−η_f). Lumped UA-NTU — for circuiting, two-phase distribution and pressure drop use a segmented coil simulator. Validate against a reference coil's measured T_e/T_c before trusting on a new design.
